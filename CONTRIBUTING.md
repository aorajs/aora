# Aora Contributing Guide

Loving `Aora` and want to get involved? Thanks!

## Set up

```bash
# Clone this repo
$ git clone git@github.com:aorajs/aora.git
$ cd aora
# Install Dependencies
$ pnpm
# Compile Src
$ pnpm build
# Start dev server for boilerplate example
$ cd examples/simple
$ pnpm dev
```

## Test

```bash
$ pnpm test
```

## Release

```bash
$ npm run release
```

## Examples

```bash
$ cd examples/simple
# Must start with pnpm
$ pnpm dev
```
